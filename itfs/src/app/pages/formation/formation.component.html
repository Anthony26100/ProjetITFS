Cloud

Nov 11

Cette formation présente différentes technologies de cloud computing et apporte de solides connaissances sur les concepts et les technologies nécessaires pour être efficace avec le cloud. Les connaissances acquises vous permettront de déterminer si les dernières technologies de cloud sont adaptées aux besoins de votre organisation d'un point de vue métier et technique, de choisir les bons fournisseurs et de planifier et mettre en œuvre une stratégie d'adoption du cloud.



Vous apprendrez à :
  • Approvisionner IaaS public avec Amazon AWS, Windows Azure, Google Cloud Platform et IaaS privé avec OpenStack.
  • Créer et déployer des applications cloud et développer une stratégie de mise en oeuvre efficace.
  • Créer et sécuriser des applications et services avec PaaS.
  • Exploiter SaaS pour optimiser le coût et les ressources.

À qui s'adresse cette formation ?

À toute personne souhaitant exploiter les avantages des technologies cloud computing.

Prérequis :

n'est pas nécessaire d'avoir une expérience du cloud computing pour cette formation. De bonnes connaissances des technologies de l'lnternet, de Microsoft Windows et des applications Web sont utiles mais pas requises.
